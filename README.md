# Food Delivery Platform

Microservices-based food delivery platform built using Java and Spring Boot.

## Tech Stack
- Java 17
- Spring Boot
- Spring Cloud Gateway
- Eureka Service Registry
- JWT Authentication
- MongoDB
- MySQL
- Kafka 

## Services
- API Gateway
- User Service
- Restaurant Service
- Order Service 
-payment service

## How to Run
1. Start Eureka Server
2. Start API Gateway
3. Start individual services
